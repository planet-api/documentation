Planet API

Planet API is a 2D planet sprite generator. Our internal process procedurally generate surfaces, map them to a true sphere, to finally render a planet at 1080x1080 with transparent background. Lighting and solar effect can be added via one of the SDKs.

Python SDK

Unlock the full potential of the API with the Python SDK. Get sprites programmatically and apply directly masks, lightning and shadow effects.

Pre-generated packs

You can directly get pre-generated sets of planets sprites from Itch.io. Many of them are free !
